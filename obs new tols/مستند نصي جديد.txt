
لا يوجد حتى الآن إضافة رسمية أو موثوقة (Plugin) للـ OBS تضمن دعم التمرير البرمجي (scroll animation) بشكل كامل وسلس داخل الـ Browser Source.
التحكم في التمرير داخل الـ Browser Source يعتمد بشكل أساسي على محرك المتصفح المدمج في OBS (Chromium)، وليس على إضافات OBS نفسها.
التفاصيل التقنية:
1. كيف يعمل الـ Browser Source في OBS؟
هو عبارة عن متصفح Chromium مدمج (نسخة خاصة).
أي كود جافاسكريبت أو CSS يعمل في Chrome غالبًا سيعمل في OBS، لكن مع بعض القيود (خاصة في الإصدارات القديمة).
2. هل هناك Plugins أو إضافات تدعم التمرير؟
لا يوجد حتى الآن إضافة رسمية في موقع إضافات OBS أو GitHub تتيح دعم التمرير البرمجي أو تحل مشاكل scroll animation بشكل مباشر.
معظم الإضافات تركز على التحكم في المشاهد، الفلاتر، المؤثرات، أو التكامل مع منصات البث، وليس على تحسين محرك المتصفح نفسه.
3. ما الذي يمكن فعله؟
الحلول كلها تدور حول:
تحديث OBS لأحدث إصدار (للحصول على Chromium الأحدث).
التأكد من إعدادات CSS وHTML كما شرحنا سابقًا.
استخدام متصفح خارجي (Window Capture) إذا كان التمرير البرمجي ضروري جدًا ولا يعمل في الـ Browser Source.

OBS Plugins يمكن تطويرها بلغات مثل ++C (الأكثر دعمًا) أو Python (عبر OBS Scripting).
الـ Browser Source في OBS مبني على Chromium Embedded Framework (CEF).
أي تحسين في التمرير أو دعم خصائص جديدة يتطلب إما:
تعديل إعدادات CEF المدمج.
أو تطوير مصدر متصفح جديد (Custom Browser Source) بخصائصك الخاصة.

2. السيناريو الأول: تطوير Plugin يحسن التمرير في الـ Browser Source الحالي

قارنة بين الخيارات
1. تطوير Plugin رسمي للـ OBS (C++)
الاحترافية: الأعلى على الإطلاق.
الدقة: تحكم كامل في كل خصائص الـ Browser Source، ويمكنك تعديل إعدادات Chromium المدمج، حقن جافاسكريبت، أو حتى تطوير مصدر متصفح جديد بالكامل.
النتيجة:
يعمل على كل الأجهزة بدون حلول خارجية.
يمكن توزيعه كإضافة رسمية أو خاصة.
يحتاج خبرة ++C وOBS API وCEF.
عيوب:
التطوير معقد ويحتاج وقت وتجربة.
كل تحديث لـ OBS قد يتطلب تحديث الإضافة.
2. تطوير تطبيق متصفح خارجي (Electron/CEF/WebView2) + Window Capture
الاحترافية: عالية جدًا (خاصة إذا كان التطبيق مصمم خصيصًا للبث).
الدقة: تحكم كامل في كل خصائص التمرير، دعم أحدث معايير CSS/JS، تحديث مستقل عن OBS.
النتيجة:
يعمل التمرير السلس 100% (لأنك تستخدم محرك متصفح حديث).
لا تعتمد على مشاكل OBS الداخلية.
يمكن تخصيص التطبيق ليكون فيه إعدادات خاصة بك.
عيوب:
يحتاج تشغيل تطبيق إضافي بجانب OBS.
Window Capture قد يحتاج ضبط دقيق للشفافية والحواف.
